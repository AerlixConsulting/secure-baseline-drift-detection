## Adding Assets

Place images referenced in documentation here. Preferred formats:

- **SVG** — scalable diagrams (recommended)
- **PNG** — screenshots and raster images
- **Mermaid** — architecture diagrams (embedded in Markdown, not stored here)

Reference assets in Markdown with relative paths:

```markdown
![Architecture Diagram](../assets/drift-detection-overview.svg)
```
